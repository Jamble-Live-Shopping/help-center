# Alterar ou Corrigir Sua Etiqueta de Envio

## O que você vai aprender

Este guia explica o que fazer se você precisar alterar ou corrigir uma etiqueta de envio na Jamble — se o endereço está errado, se precisa regenerar a etiqueta, ou se quer adicionar um número de rastreio externo.

## Antes de começar

Você precisa de:
- Uma conta de vendedor aprovada na Jamble
- Uma venda concluída com uma etiqueta já gerada (ou pronta para gerar)

## Como as etiquetas funcionam na Jamble

Após uma venda, seus pedidos são agrupados por comprador em **pacotes (bundles)**. Cada pacote recebe uma etiqueta de envio. Você gera a etiqueta no painel pós-show, e ela é criada como um PDF que você pode visualizar e imprimir.

Uma vez gerada, a etiqueta inclui:
- Endereço do remetente (seu endereço de envio)
- Endereço do destinatário (endereço do comprador)
- Um número de rastreio
- Um código de barras dos Correios

## Passo a passo

### Corrigir o endereço antes de gerar a etiqueta

Se você perceber que seu endereço está errado **antes** de gerar a etiqueta, pode corrigi-lo diretamente no painel.

**Passo 1:** Abra o painel pós-show do seu show

**Passo 2:** Seu endereço de envio aparece no topo. Se estiver errado, toque para editá-lo

**Passo 3:** Atualize seu endereço e salve. O endereço corrigido será usado quando você gerar as etiquetas

### Gerar uma nova etiqueta

Se você ainda não gerou a etiqueta para um pacote:

**Passo 1:** Abra o painel pós-show e encontre o pacote do comprador

**Passo 2:** Toque no botão para gerar a etiqueta. O sistema criará uma etiqueta dos Correios usando o serviço Melhor Envio

**Passo 3:** Uma vez gerada, o pacote mostra a opção de abrir a etiqueta. Toque para visualizá-la

### Visualizar e imprimir a etiqueta

**Passo 1:** No detalhe do pacote, toque no botão da etiqueta

**Passo 2:** A tela **Shipping Label** abre, mostrando sua etiqueta como PDF. Se o envio inclui uma declaração de conteúdo, você verá duas abas:

```
┌─────────────────────────────────┐
│  Shipping Label                 │
│                                 │
│  [ Label ] [ Content Declaration]│
│                                 │
│  ┌───────────────────────────┐  │
│  │                           │  │
│  │    [Visualização do PDF]  │  │
│  │                           │  │
│  └───────────────────────────┘  │
│                                 │
│  ┌───────────────────────────┐  │
│  │         Print             │  │
│  └───────────────────────────┘  │
└─────────────────────────────────┘
```

**Passo 3:** Toque em **Print** para exportar o PDF. Isso abre o menu de compartilhamento onde você pode imprimir diretamente, salvar em Arquivos, ou enviar para outro dispositivo

### Usar um número de rastreio externo

Se você gerou sua própria etiqueta dos Correios fora da Jamble (por exemplo, diretamente no site dos Correios ou por outro serviço), pode adicionar o número de rastreio manualmente:

**Passo 1:** Na tela do pedido/pacote, procure a opção de adicionar rastreio externo

**Passo 2:** Digite o número de rastreio ou envie uma foto da etiqueta (o app pode extrair o número de rastreio da imagem)

**Passo 3:** O sistema vinculará o número de rastreio ao pedido, para que você e o comprador possam acompanhar o envio

## Dicas importantes

- **Sempre confira seu endereço antes de gerar etiquetas.** Corrigir depois requer contato com o suporte
- **Gere etiquetas para todos os pacotes de uma vez.** No painel, você pode gerar etiquetas para todos os compradores do show com um toque — é mais rápido do que fazer uma por uma
- **Imprima as etiquetas o mais rápido possível.** Não espere muito após gerar — envie rapidamente para manter os compradores satisfeitos
- **Declarações de conteúdo** são obrigatórias para certos envios e são geradas automaticamente junto com sua etiqueta
- **Guarde o número de rastreio.** É sua prova de envio em caso de disputa

## Perguntas frequentes

**Posso regenerar uma etiqueta depois que ela foi criada?**
Entre em contato com o suporte se precisar regenerar uma etiqueta. Uma vez que a etiqueta é comprada pelo sistema, não pode ser automaticamente regenerada pelo app.

**E se o endereço do comprador estiver errado na etiqueta?**
Se o comprador forneceu um endereço incorreto, entre em contato com o suporte. Eles podem coordenar com o comprador para obter o endereço correto e emitir uma nova etiqueta se necessário.

**Posso usar minhas próprias etiquetas dos Correios ao invés das da Jamble?**
Sim. Você pode gerar sua própria etiqueta e adicionar o número de rastreio externo ao pedido. Isso é útil se você tem uma conta empresarial nos Correios ou prefere um nível de serviço diferente.

**O que é a aba Content Declaration?**
Alguns envios requerem uma declaração de conteúdo — um documento que descreve o que há dentro do pacote. Quando necessária, é gerada automaticamente junto com sua etiqueta de envio. Imprima os dois documentos.

**Gerei uma etiqueta mas agora preciso cancelar o pedido. O que acontece?**
Entre em contato com o suporte. Se a etiqueta ainda não foi usada (o pacote não foi escaneado pelos Correios), a etiqueta pode ser anulada e o reembolso processado.
