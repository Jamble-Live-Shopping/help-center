# Variants for Sellers — Sizes and Colors

## What you'll learn

This guide explains how sizes and colors work when listing products on Jamble. You'll learn how to select the right size for each product, how the size options change based on the category you pick, and how to choose colors from the available options.

## Before you start

You need:
- An approved seller account on Jamble
- A show with products to list

## How sizes and colors work on Jamble

Each product on Jamble has **one size** and **one color**. When you create a listing, you pick the size and color that match your item.

If you have the same item in multiple sizes or colors (for example, a t-shirt in S, M, and L), you create **one listing per size**. Each listing is a separate product with its own price, photos, and details.

**Size and color are optional** when listing products for a show. They're strongly recommended because buyers use size and color to decide whether to purchase — but your listing will save without them.

## Selecting a size

Size options depend on the **category** you select. Here's how it works:

### Step 1: Select a category first

When creating a listing, select the category before the size. The category uses a three-level hierarchy:

1. **Gender** — Men or Women
2. **Category** — Clothes, Shoes, Accessories, or Beauty
3. **Subcategory** — the specific item type (e.g., T-Shirts and Tops, Sneakers, Jeans)

Once you select all three levels, the **Size** field appears automatically with the right size options for your selection.

```
┌─────────────────────────────────┐
│  Category Selection             │
│                                 │
│  Gender:      [Women ▼]         │
│  Category:    [Shoes ▼]         │
│  Subcategory: [Sneakers ▼]     │
│                                 │
│  ↓ Size field appears:          │
│                                 │
│  Size: [US 7 ▼]                │
│                                 │
└─────────────────────────────────┘
```

**If you don't select a category, the size field won't appear.** That's by design — Jamble needs to know what kind of item you're selling to show you the right size options.

### Step 2: Pick the size

Tap the Size field and you'll see a list of sizes appropriate for your item type. Tap one to select it.

Plus letter sizes: XXS, XS, S, M, L, XL, XXL, XXXL, 4XL, 5XL

**Bottoms — Women (pants, jeans, shorts, skirts, leggings):**

**Men's shoes:**

**Accessories and Beauty:**

Every category also includes a **"One Size"** option (for items that don't have specific sizing) and an **"Others"** option (for sizes not in the list).

### What happens if I change the category?

If you change the category after selecting a size, the size resets automatically. You'll need to pick a new size from the updated options. This prevents mismatched sizes (like having a shoe size on a t-shirt listing).

## Selecting a color

Color works differently from size — the same **28 color options** are available for every product, regardless of category.

Tap the Color field to see the full color grid. Each color is shown as a circle with its name. Tap one to select it.

The 28 available colors are:

| | | | |
|---|---|---|---|
| Black | White | Grey | Silver |
| Beige | Cream | Apricot | Yellow |
| Mustard | Gold | Orange | Coral |
| Rose | Pink | Red | Maroon |
| Mauve | Purple | Brown | Tan |
| Mint | Turquoise | Light Blue | Blue |
| Navy | Green | Dark Green | **Multi** |

**"Multi"** is for items with multiple colors — like a tie-dye shirt, a patterned dress, or a multicolored sneaker.


You can tap **"CLEAR"** to remove your color selection if you want to leave it empty.

## Handling multiple sizes of the same item

Since each product has exactly one size and one color, here's how to handle items in multiple sizes:

**Scenario:** You have a Nike Air Max in sizes 40, 41, and 42.

**What to do:** Create 3 separate listings — one for each size. You can speed this up by:
1. Creating the first listing with all the details
2. Using **Clone** to duplicate it (see [Cloning Items](7-cloning-items.md))
3. Editing only the size and title on each clone

This takes about 30 seconds per additional size once you've created the first listing.

## Important tips

- **Select a category before choosing a size.** The size options only appear after you select Gender → Category → Subcategory. If you don't see the Size field, it's because you haven't completed the category selection
- **Include the size in your title.** Buyers often scan titles quickly during shows. "Nike Air Max 90 - 42" is immediately clear. Don't rely on the size field alone
- **Use "One Size" for universal items.** Bags, hats, and most accessories don't have specific sizes. Choose "One Size" to indicate this
- **Use "Multi" for multicolored items.** Don't pick just one of the colors — if the item has multiple prominent colors, select "Multi"
- **Be precise with shoe sizes.** Brazilian buyers know their sizes but conventions vary. If your shoes are European sizes, mention it in the title: "Nike Dunk - EUR 42 / BR 40"
- **Clone for efficiency.** If you're listing the same item in 5 sizes, create one listing, clone it 4 times, and just change the size on each. It's much faster than creating 5 from scratch

## Common questions

**Are size and color required?**
When listing for a show (Quick Upload), they're optional. They're recommended because they help buyers decide, but your listing will save without them.

**Can I add a custom size not in the list?**
You can select "Others" if your size isn't available in the preset options. Mention the specific size in your title or description (e.g., "Size: EUR 46 - see description").

**Can I select multiple colors for one product?**
No. Each product has one color. If your item has multiple colors, select **"Multi"** — that's exactly what it's for.

**Can one listing have multiple sizes (like S, M, L)?**
No. Each listing has exactly one size. If you have the same item in multiple sizes, create one listing per size. Use the Clone feature to speed this up.

**Why don't I see the Size field?**
You need to select a full category first (Gender → Category → Subcategory). The Size field only appears after all three levels are selected.

**Do buyers see the size and color during a show?**
Yes. The size is displayed on the product during the live show (unless it's set to "Others" or left empty). Color information is also visible in the product details.
