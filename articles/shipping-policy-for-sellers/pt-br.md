# Política de Envio para Vendedores

## O que você vai aprender

Este guia cobre as regras e expectativas de envio para vendedores na Jamble. Entender essas políticas ajuda a evitar problemas, manter seus compradores satisfeitos e manter uma boa reputação como vendedor.

## O básico

Quando um comprador adquire um produto durante seu show, você é responsável por:

1. **Gerar uma etiqueta de envio** no painel pós-show
2. **Embalar o item corretamente** para evitar danos durante o transporte
3. **Levar o pacote** a uma agência dos Correios
4. **Fornecer rastreio** para que o comprador possa acompanhar o pedido

Todos os envios na Jamble vão pelos **Correios**. As etiquetas são geradas pelo serviço Melhor Envio integrado ao app.

## Prazo de envio

Após uma venda, você deve enviar o mais rápido possível. Aqui estão as expectativas:

| Etapa | Prazo |
|-------|-------|
| Confirmar a venda | Dentro de 3 dias após a venda |
| Gerar a etiqueta de envio | O mais rápido possível após confirmar |
| Embalar e levar o pacote | Dentro de **10 dias** após confirmar a venda |
| Comprador recebe o item | Depende do prazo de entrega dos Correios (varia por distância) |

**Você tem 10 dias para enviar após confirmar uma venda.** Se não enviar dentro desta janela, o pedido muda para status "Ship Timeout" e pode ser automaticamente cancelado e reembolsado. Envie o mais cedo possível — não espere até o prazo final.

## O que o comprador paga

O comprador paga o frete no checkout. O custo é calculado automaticamente com base em:

- O **perfil de envio** que você atribuiu ao produto (determina tamanho/peso do pacote)
- Seu **endereço de envio** (origem)
- O **endereço do comprador** (destino)
- Sua **configuração de velocidade** (Cheapest ou Priority)
- Se o **seguro do pacote** está ativado

Você nunca paga o custo do frete dos seus ganhos. O comprador cobre integralmente.

## Etiquetas de envio

### Como gerar etiquetas

Após seu show, vá ao painel pós-show. Você verá suas vendas organizadas por comprador em pacotes. Para cada pacote, você pode:

- **Gerar uma etiqueta** — cria uma etiqueta dos Correios com rastreio
- **Gerar todas as etiquetas de uma vez** — processa etiquetas para todos os compradores do show

### O que inclui uma etiqueta

Cada etiqueta contém:
- Endereços do remetente e destinatário
- Número de rastreio e código de barras dos Correios
- Declaração de conteúdo (quando necessária)

### Opções de formato

Você pode escolher seu formato preferido em **Settings > Shipping Preferences**:
- Half Page 8.5 x 11 (padrão)
- Full Page 8.5 x 11
- Thermal 4 x 6

## Requisitos de embalagem

- **Use embalagem apropriada.** O pacote deve corresponder ao perfil de envio selecionado. Um perfil "Roupas leves" não deve ser usado para uma jaqueta pesada
- **Proteja o item.** Use plástico bolha, papel de seda ou outro preenchimento para evitar danos durante o transporte
- **Feche o pacote com segurança.** Use fita adesiva para garantir que o pacote não abra durante o manuseio
- **Cole a etiqueta de forma clara.** O código de barras e os endereços devem estar totalmente visíveis e não obstruídos

## Rastreamento

Toda etiqueta gerada pela Jamble inclui um número de rastreio. Tanto você quanto o comprador podem acompanhar o envio.

- O rastreamento é atualizado automaticamente conforme o pacote se move pela rede dos Correios
- O comprador recebe notificações quando o status do pacote muda

Se você usar uma etiqueta externa, certifique-se de adicionar o número de rastreio ao pedido no app para que o comprador possa acompanhar.

## Cancelamentos e não-envio

### Cancelamento pelo vendedor

Se você precisa cancelar um pedido (item perdido, danificado, impossibilidade de envio), pode fazer pelo app. Você será solicitado a selecionar um motivo:

- Unable to ship the item
- Item lost
- Item damaged
- Buyer requested cancellation
- Other reasons

**Aviso:** Cancelamentos abusivos ou repetidos podem levar a restrições temporárias ou suspensão da conta.

### O que acontece se você não enviar

Se você não gerar uma etiqueta e enviar o item dentro do prazo esperado:
- O comprador pode solicitar um cancelamento
- O pedido pode ser automaticamente cancelado
- Sua reputação como vendedor pode ser negativamente afetada

## Nota Fiscal

Para vendedores registrados como empresa (com CNPJ), a Jamble suporta geração de **Nota Fiscal**:

- **Geração automática** — se seu certificado está registrado, o sistema pode gerar a nota automaticamente
- **Adição manual** — você pode adicionar uma nota fiscal manualmente pelo detalhe do pacote
- **Isenção** — se você não é obrigado a emitir Nota Fiscal, pode marcar o pacote como isento

## Dicas importantes

- **Envie dentro do prazo.** Esta é a regra mais importante. Atrasos prejudicam sua reputação e frustram compradores
- **Escolha o perfil de envio correto para cada produto.** Um perfil subdimensionado pode fazer o custo real exceder o que foi cobrado, causando problemas
- **Embale os itens com cuidado.** Itens danificados levam a devoluções, reembolsos e avaliações ruins
- **Mantenha seu endereço de envio atualizado.** Se você se mudar, atualize seu endereço em Settings imediatamente — afeta os cálculos de custo de frete
- **Comunique-se com compradores se houver atraso.** Se algo surgir, avise o comprador pelo chat do app. Transparência faz muita diferença

## Perguntas frequentes

**Quem paga o frete?**
O comprador sempre paga o frete. O custo é calculado automaticamente e adicionado no checkout.

**Posso usar outra transportadora além dos Correios?**
Atualmente, todos os envios da Jamble são processados pelos Correios. Se precisar usar outra transportadora, pode gerar sua própria etiqueta externamente e adicionar o número de rastreio manualmente.

**E se um comprador reclamar de frete caro?**
Os custos são calculados com base no perfil de envio, distância e velocidade. Verifique se está usando o perfil correto — um perfil superdimensionado vai aumentar os custos desnecessariamente. Se os custos parecerem incorretos, entre em contato com o suporte.

**Quantos dias tenho para enviar?**
Você tem **10 dias** após confirmar uma venda para enviar o item. Se não enviar dentro desta janela, o pedido pode ser automaticamente cancelado e o comprador reembolsado.

**O que acontece se eu cancelar muitos pedidos?**
Cancelamentos repetidos podem resultar em restrições temporárias na sua conta. Cancele apenas quando genuinamente necessário.

**Preciso emitir Nota Fiscal?**
Se você vende como empresa (com CNPJ) e atende certos critérios, pode precisar emitir Nota Fiscal. O app vai avisá-lo quando necessário. Vendedores pessoa física (apenas CPF) geralmente são isentos.
