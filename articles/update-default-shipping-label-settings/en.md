# Update Default Shipping Label Settings

## What you'll learn

This guide explains how to update your default label settings — the format and insurance preference that apply to every new shipping label you generate. These defaults save you time by automatically applying your preferences to all future orders.

## Before you start

You need:
- An approved seller account on Jamble
- To know which label format your printer supports

## What are default label settings?

Your default label settings are preferences that apply automatically to every new shipping label. Instead of configuring each order individually, you set them once and they apply to all future labels.

There are two default settings:

1. **Label format** — the file type of your shipping labels (PDF, thermal, etc.)
2. **Package insurance** — whether to add insurance on high-value shipments

## Step by step

### Step 1: Open Shipping Preferences

Go to your profile → **Settings** → **Shipping Preferences**.

![Shipping Preferences form with address, label format, and insurance options](./assets/mockups/update-default-shipping-label-settings__shipping-preferences__en.png)

### Step 2: Update the label format

Choose the format that matches your printer:

| Format | Code | When to use |
|--------|------|-------------|
| **Half-page PDF** | PDF | You have a regular printer. This is the default |
| **Full-page PDF** | PDF_SINGLE_8X11 | You prefer a larger label on a full sheet |
| **4×6 Thermal Label** | PDF_4x6 | You have a thermal label printer |

Select the option you want. The arrow (←) shows your current selection.

### Step 3: Save your changes

Tap **Save** at the bottom of the screen. Your new defaults are now active.

All labels generated after this point will use your updated settings. Previously generated labels are not affected.

## When to update your defaults

- **You bought a thermal printer** — Switch label format to 4×6 Thermal Label
- **You sell high-value items** — Turn on package insurance
- **Your current printer broke and you're using a different one** — Switch to the format your new printer supports
## Important tips

- **Defaults apply to ALL future labels.** Every new label will use these settings until you change them again
- **Previously generated labels are not affected.** If you switch from PDF to thermal, your existing labels stay as PDFs
- **You can change defaults anytime.** There's no limit on how often you update your settings
- **Test your new format.** After switching label formats, generate a test label on your next order to make sure it prints correctly before committing to a large batch
- **Insurance only applies to high-value shipments.** The insurance is activated on shipments where the total item value exceeds R$ 500

## Common questions

**Will changing my defaults affect orders I've already confirmed?**
No. Only new labels generated after the change will use the new settings.

**Can I set different defaults for different types of orders?**
No. The defaults apply to all orders. If you need a different format for a specific order, change the default before generating that label, then change it back.

**I don't know which label format to pick. What should I do?**
Start with **Half-page PDF**. It works with any regular printer and is the most common choice. If you later get a thermal printer, switch to 4×6 Thermal Label.

**When does insurance apply?**
Insurance is automatically added to shipments where the total item value exceeds R$ 500. For shipments below this threshold, the insurance toggle has no effect on cost.
