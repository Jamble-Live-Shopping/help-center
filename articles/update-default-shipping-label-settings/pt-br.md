# Atualize as Configurações Padrão de Etiqueta

## O que você vai aprender

Este guia explica como atualizar suas configurações padrão de etiqueta — o formato e a preferência de seguro que se aplicam a cada nova etiqueta que você gera. Esses padrões economizam tempo aplicando automaticamente suas preferências a todos os pedidos futuros.

## Antes de começar

Você precisa de:
- Uma conta de vendedor aprovada na Jamble
- Saber qual formato de etiqueta sua impressora suporta
## O que são configurações padrão de etiqueta?

Suas configurações padrão de etiqueta são preferências que se aplicam automaticamente a cada nova etiqueta de envio. Ao invés de configurar cada pedido individualmente, você define uma vez e elas se aplicam a todas as etiquetas futuras.

Existem duas configurações padrão:

1. **Formato da etiqueta** — o tipo de arquivo das suas etiquetas de envio (PDF, térmica, etc.)
2. **Seguro do pacote** — se adiciona seguro em envios de alto valor

## Passo a passo

### Passo 1: Abra Shipping Preferences

Vá ao seu perfil → **Settings** → **Shipping Preferences**.

### Passo 2: Atualize o formato da etiqueta

Escolha o formato que corresponde à sua impressora:

| Formato | Quando usar |
|---------|------------|
| **PDF meia página** | Você tem uma impressora comum. Este é o padrão |
| **PDF página inteira** | Você prefere uma etiqueta maior em uma folha inteira |
| **Etiqueta térmica 4×6** | Você tem uma impressora de etiqueta térmica |

Selecione a opção que deseja. A seta (←) mostra sua seleção atual.

### Passo 3: Salve suas alterações

Toque em **Save** na parte inferior da tela. Seus novos padrões estão ativos.

Todas as etiquetas geradas a partir deste ponto usarão suas configurações atualizadas. Etiquetas geradas anteriormente não são afetadas.

## Quando atualizar seus padrões

| Situação | O que mudar |
|----------|------------|
| Você comprou uma impressora térmica | Mude o formato para Etiqueta Térmica 4×6 |
| Você vende itens de alto valor | Ative o seguro do pacote |
| Sua impressora atual quebrou e está usando outra | Mude para o formato que sua nova impressora suporta |

## Dicas importantes

- **Os padrões se aplicam a TODAS as etiquetas futuras.** Toda nova etiqueta usará essas configurações até você mudá-las novamente
- **Etiquetas já geradas não são afetadas.** Se trocar de PDF para térmica, suas etiquetas existentes continuam como PDFs
- **Você pode mudar os padrões a qualquer momento.** Não há limite de quantas vezes atualiza
- **Teste o novo formato.** Após mudar o formato, gere uma etiqueta de teste no próximo pedido para garantir que imprime corretamente antes de se comprometer com um lote grande
- **Seguro só se aplica a envios de alto valor.** O seguro é ativado em envios onde o valor total dos itens ultrapassa R$ 500

## Perguntas frequentes

**Mudar meus padrões vai afetar pedidos que já confirmei?**
Não. Apenas novas etiquetas geradas após a mudança usarão as novas configurações.

**Posso definir padrões diferentes para tipos diferentes de pedidos?**
Não. Os padrões se aplicam a todos os pedidos. Se precisar de um formato diferente para um pedido específico, mude o padrão antes de gerar aquela etiqueta, depois mude de volta.

**Não sei qual formato de etiqueta escolher. O que devo fazer?**
Comece com **PDF meia página**. Funciona com qualquer impressora comum e é a escolha mais popular. Se depois comprar uma impressora térmica, mude para Etiqueta Térmica 4×6.

**Quando o seguro é aplicado?**
O seguro é automaticamente adicionado a envios onde o valor total dos itens ultrapassa R$ 500. Para envios abaixo desse valor, a opção de seguro não tem efeito no custo.
