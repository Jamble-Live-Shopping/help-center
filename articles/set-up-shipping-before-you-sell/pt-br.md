# Configure o Envio Antes de Vender

## O que você vai aprender

Este guia explica como configurar suas preferências de envio na Jamble para que você possa começar a vender. Você vai aprender a configurar seu endereço de envio, escolher o formato de etiqueta e entender como os custos de frete funcionam para seus compradores.

## Antes de começar

Você precisa de:
- Uma conta de vendedor aprovada na Jamble
- Seu endereço de envio (com CEP)
- Saber qual formato de etiqueta você quer (impressora comum ou impressora térmica)

## Passo a passo

### Passo 1: Abra suas configurações de envio

Vá ao seu perfil, depois toque em **Configurações**. Role para baixo e toque em **Shipping Preferences**.

```
┌─────────────────────────────────┐
│  Configurações                  │
│                                 │
│  Configurações do Perfil        │
│  Configurações de Pagamento     │
│  ▶ Shipping Preferences         │
│  Configurações de Notificação   │
│                                 │
└─────────────────────────────────┘
```

Isso abre a tela onde você configura tudo relacionado ao envio.

### Passo 2: Defina seu endereço de envio

Seu endereço de envio é de onde os pacotes são retirados ou onde você os entrega. Este endereço também é usado para calcular os custos de frete para os compradores.

Preencha:
- **CEP** (código postal) — quando você digitar seu CEP, o app preenche automaticamente rua, bairro, cidade e estado
- **Número** — o número da sua casa ou prédio
- **Complemento** (opcional) — se você mora em apartamento

```
┌─────────────────────────────────┐
│  Endereço de Envio              │
│                                 │
│  CEP: _____-___                 │
│  Rua: [preenchido auto]         │
│  Número: ____                   │
│  Complemento: ____ (opcional)   │
│  Bairro: [preenchido auto]      │
│  Cidade: [preenchido auto]      │
│  Estado: [preenchido auto]      │
│                                 │
│  ┌───────────────────────────┐  │
│  │     Salvar Endereço       │  │
│  └───────────────────────────┘  │
└─────────────────────────────────┘
```

**Por que isso importa:** Seu endereço determina qual agência dos Correios é a mais próxima para entregas, e afeta o custo de frete que os compradores veem no checkout.

### Passo 3: Escolha o formato da etiqueta de envio

A Jamble gera etiquetas de envio para cada pedido. Você precisa escolher o formato que funciona com sua impressora.

Você tem três opções:

- **PDF meia página** — a opção padrão. Funciona com qualquer impressora comum. Imprime em meia folha de papel
- **PDF página inteira** — usa uma folha inteira. Bom se você prefere etiquetas maiores e mais fáceis de ler
- **Etiqueta térmica 4×6** — para vendedores que têm uma impressora térmica de etiquetas (como Zebra ou similar). São etiquetas adesivas, não precisa cortar

```
┌─────────────────────────────────┐
│  Formato da Etiqueta            │
│                                 │
│  ○ PDF meia página (padrão)     │
│  ○ PDF página inteira           │
│  ○ Etiqueta térmica 4×6         │
│                                 │
└─────────────────────────────────┘
```

**Se não tem certeza de qual escolher:** Comece com PDF meia página. Funciona com qualquer impressora e é a escolha mais comum.

### Passo 4: Entenda como os custos de frete funcionam

Algo importante que muitos vendedores novos perguntam: **você não define os preços de frete.** Veja como funciona:

1. Quando você lista um produto, você atribui um **perfil de envio** — uma categoria de tamanho/peso (por exemplo, "Roupas leves" ou "Roupas padrão")
2. Quando um comprador adquire seu produto, a Jamble calcula automaticamente o custo do frete com base em:
   - O perfil de envio (tamanho e peso do pacote)
   - Seu endereço (de onde o pacote sai)
   - O endereço do comprador (para onde o pacote vai)
3. **O comprador paga o custo do frete** no checkout — é adicionado ao preço do produto

Você nunca paga o frete dos seus ganhos. O comprador sempre cobre o custo de envio.

### Passo 5: Entenda os perfis de envio

Ao listar produtos, você escolhe um perfil de envio que corresponde ao tamanho e peso do seu pacote. Aqui estão os perfis disponíveis:

| Perfil | Melhor para | Exemplos |
|--------|-------------|----------|
| XS | Cartas, itens planos | Cartas Pokémon, adesivos |
| S | Pacotes de cartas, acessórios pequenos, joias | Boosters, brincos, pulseiras |
| M | Camisetas, roupas leves | Camisetas, regatas, blusas |
| L | Roupas normais, calçados | Camisas, vestidos, tênis |
| XL | Jaquetas, casacos de inverno, pacotes | Casacos pesados, grandes coleções |

**Escolha o perfil que corresponde ao tamanho e peso real do seu item.** Se escolher um perfil pequeno demais, o pacote pode custar mais para enviar e causar problemas. Se escolher um grande demais, o comprador paga mais do que o necessário, o que pode desestimular compras.

## Dicas importantes

- **Configure o envio ANTES do seu primeiro show.** Se você vender algo sem ter o envio configurado, não vai conseguir gerar etiquetas e seus pedidos vão atrasar.
- **Seu CEP preenche o endereço automaticamente.** Você só precisa digitar o código postal — o app preenche o resto. É só adicionar o número.
- **Comece com etiquetas PDF meia página** se não tem certeza. Você sempre pode mudar para etiquetas térmicas depois conforme seu negócio cresce.
- **Todos os envios vão pelos Correios.** Depois de gerar uma etiqueta, leve seu pacote à agência dos Correios mais próxima.

## Perguntas frequentes

**Eu defino o preço do frete dos meus produtos?**
Não. A Jamble calcula o frete automaticamente com base no tamanho do pacote (perfil de envio), sua localização e a localização do comprador. O comprador vê o custo exato do frete no checkout.

**Quem paga o frete?**
O comprador sempre paga o frete. O custo é adicionado no checkout ao preço do produto. Seus ganhos nunca são reduzidos pelo custo de frete.

**Posso usar outra transportadora além dos Correios?**
Atualmente, todos os envios da Jamble são feitos pelos Correios. O sistema seleciona automaticamente a opção de frete mais econômica para cada pedido.

**E se eu escolher o perfil de envio errado para um produto?**
Você pode mudar o perfil de envio de qualquer produto antes que ele seja vendido. Se perceber depois de uma venda que o perfil estava errado, entre em contato com o suporte — eles podem ajudar a ajustar a etiqueta.

**Preciso de uma impressora térmica?**
Não. Uma impressora comum funciona bem com as opções PDF meia página ou página inteira. Impressoras térmicas são mais rápidas e práticas para vendedores de alto volume, mas não são obrigatórias.

**Como gero uma etiqueta de envio depois de uma venda?**
Depois que um comprador adquire seu item, vá aos seus pedidos. Cada pedido terá uma opção para gerar uma etiqueta de envio. A etiqueta é criada automaticamente — você só precisa imprimir e colar no pacote.

**Onde entrego meus pacotes?**
Leve seus pacotes etiquetados a qualquer agência dos Correios. Você pode encontrar a mais próxima no site ou app dos Correios.
