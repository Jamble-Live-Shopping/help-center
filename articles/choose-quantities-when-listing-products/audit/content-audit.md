## Copy fidelity fixes (major)

This rewrite corrected 3 faithfulness bugs present in the prior version:

1. **"pre-offer" / "pre-offers" invented copy**: the app has no such term. Replaced with "Pre-Bid" throughout prose (matches UI labels `"Enable Pre-Bid?"` and section title `"Pre-Bid"`).
2. **Wrong error-message quote**: article had `"You can not use pre-offer if you have more than one quantity"`. Actual code string is `"You can not use prebid if you have more than one quantity"` (lowercase, one word). Fixed in alt text and rendered in PNG.
3. **Incomplete toast representation**: article only quoted the subtitle. Real toast from `JambleIndicatorView` has a title line too ("Oops, something happened!"). Mockup now shows both lines.
