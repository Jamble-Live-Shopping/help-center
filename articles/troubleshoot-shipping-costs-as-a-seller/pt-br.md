# Resolver Problemas de Custo de Frete como Vendedor

## O que você vai aprender

Este guia explica como os custos de frete são calculados na Jamble, por que os custos podem parecer errados e o que você pode fazer. Entender isso ajuda a lidar com dúvidas dos compradores e evitar problemas comuns de preço.

## Como os custos de frete são calculados

Os custos de frete na Jamble são **automáticos**. Você não os define. Veja como funciona:

1. Quando você lista um produto, atribui um **perfil de envio** (ex: Roupas leves, Roupas padrão, etc.)
2. O perfil de envio determina as **dimensões e peso do pacote** para cálculo
3. Quando um comprador adquire o item, a Jamble calcula o custo de frete com base em:
   - O perfil de envio (tamanho/peso)
   - Seu endereço de envio (origem)
   - O endereço do comprador (destino)
   - Sua configuração de velocidade (Cheapest ou Priority)
   - Se o seguro do pacote está ativado

O comprador vê e paga o custo exato do frete no checkout.

## Problemas comuns de custo de frete

### "O comprador diz que o frete está caro demais"

**Por que acontece:**
- O perfil de envio pode ser maior do que o necessário para o item real
- O comprador está longe da sua localização (rotas de longa distância custam mais)
- A velocidade Priority está selecionada (mais rápido = mais caro)
- O seguro do pacote está ativado (adiciona um pequeno custo)

**O que fazer:**
1. **Verifique o perfil de envio.** Se você atribuiu "Itens volumosos" a uma camiseta, o custo de frete será muito maior do que o necessário. Mude o perfil para corresponder ao item real
2. **Verifique sua configuração de velocidade.** Se Priority está selecionado em Shipping Preferences, mude para Cheapest para reduzir custos
3. **Verifique se o seguro está ativado.** Se está e o item é de baixo valor, desativar o seguro reduz um pouco o custo
4. **Explique ao comprador** que os custos de frete são baseados no tamanho do pacote, peso e distância. Você não controla o preço — os Correios sim

### "O custo de frete parece baixo demais"

**Por que acontece:**
- O perfil de envio pode ser menor que o item real
- Isso é um problema porque o custo real nos Correios pode exceder o que foi cobrado

**O que fazer:**
- Use o perfil de envio correto para o tamanho real do item. Se você consistentemente usa perfis menores, pode ter problemas no balcão dos Correios

### "Compradores diferentes veem custos de frete diferentes para o mesmo produto"

**Por que acontece:**
Isso é normal. Os custos de frete dependem da **localização do comprador**. Um comprador na mesma cidade paga muito menos que um comprador em outro estado. Não há nada errado — é como funciona o frete baseado em distância.

### "Os custos de frete mudaram para o mesmo produto"

**Por que acontece:**
- Você pode ter mudado sua configuração de velocidade (Cheapest vs Priority)
- Você pode ter ativado ou desativado o seguro do pacote
- Os Correios podem ter atualizado suas tabelas de preço (isso acontece periodicamente)

## Perfis de envio e seu impacto no custo

O perfil de envio é o maior fator que você controla. Veja como os perfis se relacionam ao custo:

| Perfil | Custo relativo |
|--------|---------------|
| Carta | Mais baixo |
| Booster | Muito baixo |
| Acessórios leves | Baixo |
| Roupas leves | Baixo-médio |
| Roupas padrão | Médio |
| Roupas mais pesadas | Médio-alto |
| Itens volumosos | Alto |
| Pacotes pequenos | Alto |
| Pacotes médios | Muito alto |
| Pacotes grandes | Muito alto |
| Pacotes extra-grandes | Mais alto |

**A regra é simples:** Use o menor perfil em que seu item embalado realmente cabe. Não superdimensione (comprador paga demais) nem subdimensione (causa problemas nos Correios).

## Como mudar um perfil de envio

Se você perceber que um produto tem o perfil errado:

1. Abra a listagem do produto
2. Edite os detalhes do produto
3. Mude o perfil de envio para o correto
4. Salve

**Importante:** Você só pode mudar o perfil de envio antes do item ser vendido. Após uma venda, o custo de frete é travado no perfil atribuído no momento da compra.

## Dicas importantes

- **Sempre atribua o perfil de envio correto.** Esta é a causa mais comum de problemas de custo. Reserve um momento para considerar o tamanho real do item embalado
- **Cheapest é recomendado.** A menos que seus compradores queiram entrega mais rápida, Cheapest mantém os custos baixos e torna seus produtos mais atraentes
- **Lembre: você não paga o frete.** O comprador paga. Mas se os custos estão altos demais, compradores podem não comprar — então é do seu interesse manter os perfis precisos
- **Revise suas configurações periodicamente.** Se já faz tempo que não olhou suas Shipping Preferences, confira para ter certeza de que tudo ainda está correto

## Perguntas frequentes

**Posso oferecer frete grátis?**
Atualmente, os custos de frete são sempre cobrados do comprador. Vendedores não podem subsidiar ou eliminar custos de frete pelo app.

**Por que o frete para o Norte/Nordeste é mais caro?**
Os preços dos Correios são baseados em distância. Envios para regiões mais remotas naturalmente custam mais. É uma política da transportadora, não uma configuração da Jamble.

**Posso mudar o custo de frete após uma venda?**
Não. O custo de frete é calculado e cobrado no momento da compra. Não pode ser modificado depois.

**Minha configuração de velocidade afeta todos os meus produtos?**
Sim. A configuração Cheapest/Priority em Shipping Preferences se aplica a todos os seus envios. Não é possível definir velocidades diferentes para produtos diferentes.

**E se os Correios cobrarem mais no balcão do que o que estava na etiqueta?**
Se as dimensões reais do pacote diferirem significativamente do perfil de envio, os Correios podem cobrar uma diferença. Por isso é fundamental usar o perfil correto. Se isso acontecer, entre em contato com o suporte.
